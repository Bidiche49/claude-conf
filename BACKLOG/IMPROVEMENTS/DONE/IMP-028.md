# IMP-028 — tab-titles: ajouter tests pour session-tab-title.sh (SessionStart hook)

**Priorité:** Basse
**Module:** tab-titles

## Description

Le nouveau hook `session-tab-title.sh` (SessionStart, ajouté par BUG-010) n'a pas de tests automatisés.

## Tests à ajouter dans `tab-titles/tests/test-tab-titles.sh`

1. **Module disabled** — quand `tab-titles` est dans `~/.claude-conf-disabled`, le hook exit 0 sans écrire de titre
2. **Env vars présentes** — quand `CC_TAB_TITLE` et `CC_WIN_TITLE` sont set, le hook écrit les bonnes séquences OSC
3. **Env vars absentes** — quand les env vars sont vides/non définies, le hook exit 0 sans rien écrire
4. **Stdin consommé** — le hook consomme stdin (JSON input) sans erreur

## Fichiers impactés

- `tab-titles/tests/test-tab-titles.sh`
