# backlog

Universal ticketing system for Claude Code with automatic ID protection.

## What it does

- **Ticketing system** — bugs, features, improvements with structured templates and conventions
- **backlog-guard hook** — PreToolUse Write hook that blocks duplicate ticket IDs across concurrent sessions
- **Auto-generated INDEX.md** — never edited manually, regenerated by `/backlog-status`

## Structure created by `/backlog-init`

```
BACKLOG/
├── INDEX.md
├── BUGS/{PENDING,DONE}/
├── FEATURES/{PENDING,DONE}/
├── IMPROVEMENTS/{PENDING,DONE}/
└── _templates/ticket.md
```

## Install

```bash
bash install.sh
```

## Dependencies

- `claude` — Claude Code CLI
- `jq` — JSON processor (for the hook)

## Commands

| Command | Description |
|---------|-------------|
| `/backlog-init` | Initialize BACKLOG/ structure in a project |
| `/backlog-bug <desc>` | Create a bug ticket |
| `/backlog-feat <desc>` | Create a feature ticket |
| `/backlog-imp <desc>` | Create an improvement ticket |
| `/backlog-status` | Regenerate INDEX.md and show stats |

## Works best with

- **supervisor** — CTO mode for ticket planning and delegation
